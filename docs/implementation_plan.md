# おしゃれな時計Webアプリ "Aurora Time" 実装計画

## 目標
ユーザーを魅了する、プレミアムで美しいデザインの時計Webアプリを作成する。
コンセプトは「Aurora Time（オーロラ・タイム）」。ゆっくりと変化する美しいグラデーション背景と、グラスモーフィズムを取り入れたミニマルなインターフェースを特徴とする。

## 構成
- **技術スタック**: React + Vite, Vanilla CSS
- **デザイン**: オーロラ・グラデーション + グラスモーフィズム
- **機能**: テーマ切り替え、インタラクティブ背景、没入モード

## コンポーネント構成
1.  **`App.jsx`**: メインレイアウト。
2.  **`components/Clock.jsx`**: 時計表示。
3.  **`components/ThemeSwitcher.jsx`**: テーマ切り替え。
4.  **`components/Controls.jsx`**: 各種操作。
5.  **`styles/index.css`**: グローバルスタイル。
